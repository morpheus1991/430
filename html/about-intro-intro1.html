  • 퍼블리싱

    사용 기술
    • HTML 시멘틱하고 스타일링이 고려된 마크업
    • CSS 전처리기(SCSS)에 능숙하며 컴포넌트 단위로 생산성 높은 스타일링
    • javascript 라이브러리 사용, 조건부 toggle, 이벤트 위임 등이 가능
      (스코프, 호이스팅, 클로져 등 이해)
    • git 주요 개념에 대해서 이해하고 있으며 활용
      ( gui/cli 사용)
    활용 툴
    • Vscode scss 설정 및 esLint 등 확장도구 설정하여 사용
    • SourceTree 내역 병합, 커밋, 푸쉬, 풀 등의 기능을 익숙하게 사용,
      브렌치 전략수립이 가능
    • Figma,Zepelin ux/ui디자이너를 양성한 경험이 있으며,
      디자인시스템에 대해서 이해
